# Security Policy

## Reporting a Vulnerability

If you believe you've found a security vulnerability in Peatopea, please
report it privately to: cashlycash.g@gmail.com

Please include:
- A clear description of the vulnerability
- Steps to reproduce
- Affected versions
- Any exploit code or proof-of-concept (if available)

We will respond within 7 business days with next steps.

## Supported Versions

We support the latest release and the previous minor release. If you are
unsure whether a version is supported, contact the security contact above.

## Disclosure Policy

We prefer coordinated disclosure. Do not publicly disclose a vulnerability
until it has been resolved or we have agreed on a disclosure timeline.

## Fixing Vulnerabilities

Once a fix is available, we will publish release notes and notify
reporters. If the vulnerability is high-severity and a coordinated release
is necessary, follow-up instructions will be provided to affected parties.
